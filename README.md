Website Menu Hải Sản – Tiệm Hải Sản Ba Cô Gái

## Giới thiệu
Đây là website giới thiệu và hiển thị **menu hải sản tươi sống**, phục vụ cho việc:
- Xem danh sách các loại hải sản
- Xem giá bán
- Giao diện trực quan, dễ sử dụng
- Phù hợp cho cửa hàng / tiệm hải sản nhỏ

Website được xây dựng bằng **HTML, CSS và JavaScript*
- HTML5
- CSS3
- JavaScript
- Không sử dụng framework
 Hướng dẫn chạy website
### Cách 1: Mở trực tiếp
1. Tải toàn bộ source code về máy
2. Mở file `index.html`
3. Click đúp hoặc mở bằng trình duyệt (Chrome, Edge, Firefox)
   
 Thành viên nhóm
- Thành viên 1: nguyễn tấn phát
- Thành viên 2: âu nguyên khang
- Thành viên 3: nguyễn hữu đưc
